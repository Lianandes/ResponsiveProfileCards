# Responsive Profile Cards

Projeto desenvolvido para praticar a criação de cartões de perfil responsivos utilizando HTML e CSS.

## 🚀 Objetivo

O objetivo deste projeto foi aprimorar conhecimentos em desenvolvimento front-end, responsividade e estilização de componentes modernos.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Layout responsivo
- Design moderno
- Cards de perfil estilizados
- Efeitos visuais
- Organização visual dos componentes

## 📚 Conceitos praticados

- Flexbox
- Responsividade
- Media Queries
- Posicionamento de elementos
- Estilização com CSS
